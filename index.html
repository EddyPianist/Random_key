Fine-tuned vs Original Model Comparison

By Eddy

Each audio sample below is generated using both the fine-tuned version of the model and the original one. This comparison shows how conditioning and fine-tuning improve control and sound quality.

Prompt Set: piano notes across octaves | CFG = 7

The following samples compare outputs from the original model and the fine-tuned version, across a range of piano notes. This showcases how fine-tuning helps stabilize pitch control and improves consistency across different octaves.

Original Model: C6, D6, E5, G6

Fine-tuned Model: C6, D6, E5, G6

As you listen, note how the original model struggles to maintain precise pitch and timbre, especially for high or low octaves. The fine-tuned version delivers much clearer and more consistent note generation.
